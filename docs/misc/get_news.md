# Creator Portal - Get List of News
> Gives you a list of the latest Creative news visible on the Creator Portal.

URL: `https://create.fortnite.com/api/cms/v1/articles` \
Method: `GET` \
Authentication: `NO` \
Language Support: `NO`

Permissions required:
 - None
